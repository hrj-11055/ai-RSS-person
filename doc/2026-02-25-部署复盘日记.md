## 真实踩坑与收敛

- 一开始登录错了服务器，导致路径判断偏差（`/opt` 不存在）。
- timer 启动失败，本质是 unit 文件未安装到 `/etc/systemd/system/`。
- 日报服务报 `203/EXEC`，原因是 `.venv` 不存在。
- Python 运行报 `No module named yaml`，补装 `PyYAML` 后恢复。
- RSSHub 源大量失败，根因是 `localhost:1200` 无服务。
- RSSHub 部署中遇到 npm mirror 404、Node 版本兼容与构建产物缺失问题，最终通过切换官方 registry + 构建 + pm2 常驻解决。
